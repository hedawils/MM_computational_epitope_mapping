# Sequence similarity analysis

Generated by [../../scripts/sequence_similarity_analysis.py](../../scripts/sequence_similarity_analysis.py).

## Method

The 15 clones share an identical human IgG1 constant region on the heavy chain and
an isotype-matched (kappa or lambda) human constant region on the light chain, so
comparing full chains would mostly measure constant-region identity rather than
anything biologically meaningful. The script isolates the rabbit-derived variable
domains (VH, VL) by finding the longest common suffix within each chain/isotype
group — 331 aa (HC), 107 aa (LC kappa), 105 aa (LC lambda) — and stripping it.
Extracted domains: [../../data/processed/variable_domains.fasta](../../data/processed/variable_domains.fasta).

All pairwise VH-vs-VH and VL-vs-VL comparisons (105 pairs each) use a pure-Python
global (Needleman-Wunsch) alignment, linear gap penalty (match +1, mismatch -1,
gap -2). Percent identity = identical aligned positions / total alignment length.

## Outputs

- `identity_matrix_VH.csv`, `identity_matrix_VL.csv` — 15x15 %identity matrices
- `dendrogram_VH.png`, `dendrogram_VL.png` — average-linkage hierarchical clustering
  on (100 - %identity), labeled by clone and target antigen
- `similarity_summary.txt` — mean %identity for same-target vs. different-target
  clone pairs (clones with unresolved targets — HW-8, TH9-022 — excluded from this
  comparison)

## Notes

Mean same-target vs. different-target identity is close for both VH (71.0% vs.
70.0%) and VL (76.0% vs. 75.0%) — sequence similarity alone does not cleanly
separate clones by target antigen in this panel, i.e. clones against the same
target did not arise from an obviously shared lineage. The dendrograms are more
useful for spotting near-duplicate/clonally-related pairs (e.g. HW-42 and HW-1
cluster tightly in VH) than for antigen-based grouping.
