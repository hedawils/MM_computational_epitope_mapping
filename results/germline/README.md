# VH germline gene assignment

Generated by [../../scripts/assign_vh_germline.py](../../scripts/assign_vh_germline.py).

## What this is (and isn't)

These are rabbit-derived VH domains in a chimeric rabbit/human construct (see
[../../data/raw/sequences/README.md](../../data/raw/sequences/README.md)), so
human IGHV nomenclature (e.g. "VH3-23") does not apply. Instead, each clone's VH
domain was BLASTP'd against IMGT/GENE-DB's rabbit (*Oryctolagus cuniculus*) IGHV
germline gene reference set to find the closest known germline gene.

**This is a nearest-germline lookup, not a validated IgBLAST/V(D)J assignment.**
No D or J gene calling, no junction/CDR3 analysis, and the germline reference only
covers FR1 through FR3 (CDR3 and FR4 come from D/J segments and are not part of
the germline V gene) — the BLASTP hit is driven by framework + CDR1/CDR2
similarity, which is exactly what a V-gene call should be driven by, but treat the
result as "closest known germline," not a certified call.

## Method

1. Fetched the rabbit IGHV germline set from IMGT/GENE-DB
   (`GENElect?query=7.2+IGHV&species=Oryctolagus+cuniculus`), saved raw at
   [../../data/raw/germline/imgt_rabbit_IGHV_raw.txt](../../data/raw/germline/imgt_rabbit_IGHV_raw.txt)
   (67 alleles: 39 functional, 28 pseudogene).
2. Translated each germline V-REGION nucleotide sequence to protein (Biopython).
3. BLASTP (NCBI BLAST+, installed via `brew install blast`) each clone's VH
   domain (from [../../data/processed/variable_domains.fasta](../../data/processed/variable_domains.fasta))
   against the translated germline set; kept the top hit by bitscore.

## Result

12 of 15 clones' top hit is **IGHV1S69\*01**, consistent with the earlier
observation that most VH frameworks are nearly identical across this panel
(see [../sequence_similarity](../sequence_similarity/README.md)) — i.e. this
looks like a semisynthetic library built on a small number of fixed
scaffolds rather than natural repertoire diversity. HW-42 and HW-56 — the pair
flagged earlier as having a visibly different framework — both instead
top-hit **IGHV1S34\*01**. HW-101 and HW-1 top-hit two other alleles
(IGHV1S42\*01, IGHV1S26\*01) each. All top hits are in the rabbit IGHV1 family,
consistent with rabbit VH repertoires being IGHV1-dominated in vivo.
